# 2023102260-huangpengyin-theorywork4
2023102260计算机视觉理论作业
对目标图像分块后，分别用FFT 法（95% 能量最高频率）和梯度法（近似最高频率）计算每个块的主导频率，两者的一致性极差：
线性相关系数仅为 0.18，几乎无相关性；
平均绝对误差（MAE）高达 1.3911，两种方法的频率估计结果差异极大；
结论：两种方法在图像频率估计任务上不具备一致性，不能互相替代，适用场景完全不同。

（1）实验设计逻辑
对 C 罗的图像进行分块，得到若干个局部图像块（如第二张图的阶梯块就是其中一个样本）；
对每个块用两种方法计算频率：
FFT 法：对块做快速傅里叶变换，得到功率谱，提取包含 95% 能量的最高频率（全局频域特征）；
梯度法：计算块内的亮度梯度（边缘变化率），用梯度近似得到最高频率（局部空域特征）；
用散点图、误差直方图统计两种方法的结果，评估一致性。

（2）一致性差的具体表现（对应第三张图）
① 相关性散点图（Correlation = 0.1800）
横轴：FFT 法的 95% 能量频率，纵轴：梯度法的梯度频率，每个点对应一个图像块；
相关系数仅 0.18，说明两种方法的结果几乎无线性相关：
同一个图像块，FFT 算出的频率高，梯度法算出的频率可能低，反之亦然；
无法用一种方法的结果预测另一种方法的结果，完全不具备一致性。
② 误差分布直方图（MAE = 1.3911）
误差定义：Error = 梯度法频率 - FFT法频率；
平均绝对误差 1.3911，说明两种方法的频率估计结果平均相差 1.39 个频率单位，误差极大；
误差分布呈单峰右偏：大部分样本误差集中在 1~2 之间，少数样本误差超过 3，进一步验证了两种方法的一致性极差。

为什么一致性差：
FFT 法:把图像从空间域转换到频域，分析全局频率分量	图像块的全局频域特征，反映整个块的整体纹理、结构频率	全局频率估计、噪声分析、图像压缩
梯度法：计算图像块内的亮度梯度（边缘变化率），用边缘变化近似频率	图像块的局部空域特征，仅反映块内的边缘、突变信息	边缘检测、实时特征提取、局部纹理分析
